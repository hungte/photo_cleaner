# photo_cleaner
A set of tools to help cleaning photos

我在幫小朋友製作相片書時，從 Google Takeout 下載了每年的相片，但發現要從中整理出相片書需要的照片很困難。
相片書如果以最多的80頁、每頁4張照片，那就只能放 320 張，但每年的照片幾乎都是破千張的。
打開來看，會發現有很多重複的、拍模糊的、只有影片沒照片的、從LINE轉貼的、或是開了動態攝影變成 MOV的。
更糟的是 Google Photo 下載的照片常沒有日期的 EXIF，檔名有時也沒有日期，這些資訊被放在獨立的 json 裡。

為了方便整理，我需要一系列用 Python 作成的小工具。有些自己寫，有些叫 Gemini 幫忙。
這個目錄就整合了我所使用的工具。
